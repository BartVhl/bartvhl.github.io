# Bart’s Portfolio

Personal portfolio website hosted on **GitHub Pages**.  
Built with **HTML, CSS, and JavaScript**, featuring a custom component system and full **Dutch / English language support**.

## Featured projects

### **Gleenie.org**
Laravel project with a Tailwind frontend, built from a design document using reusable UI blocks.  
Focus on frontend structure, clean implementation, and consistency.

### **ICT21**
Laravel-based platform featuring a CMS built with **Filament** and a contact form powered by **Livewire**.  
This project required a deep dive into Filament resources and backend content management.

### **Gogomon**
Team-built fitness app for mobile and smartwatch platforms, primarily developed in **.NET MAUI**.  
Included database and API planning, platform research, a **Wear OS** implementation, and strong attention to **GDPR compliance**.

### **Deckbuilder**
MVC web application created to explore **REST APIs**, request/result models, and loose coupling.  
Also provided hands-on experience with sessions and **JWT-based authentication**.
